# 两客一危

> 一个Vue.js项目

## 构建步骤

``` bash
# 安装依赖关系
npm install

# 在8080端口进行热加载
npm run dev

# 打包上线
npm run build
```

## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + axios
